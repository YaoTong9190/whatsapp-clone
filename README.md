## whatsapp fullstack clone project

### MERN Stack WhatsApp Clone

WhatsApp Clone built with MERN (MongoDB, ExpressJS, ReactJS, NodeJS)

### Tech Stack

- React js
- Firebase for deploy
- Firebase Google Authentication
- React Hooks
- React Context API
- React Router
- styled-components
- material-ui icons
- Pusher middleware

### Use

```
npm install && npm start
```
